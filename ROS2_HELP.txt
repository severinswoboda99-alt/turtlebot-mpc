ROS2 HELP

+Remove old build artifacts
$ rm -rf build/ install/ log/

+Before working in a terminal you need to setup Jazzy: 
$ source /opt/ros/jazzy/setup.bash

$ source /opt/ros/jazzy/setup.bash && source install/setup.bash

+Rebuild (f.e. after editing code) with:
$ cd ~/xxx
$ colcon build
$ colcon build --packages-select mpc-tracker
$ colcon build --symlink-install
$ source install/setup.bash

+Important Check-Ups:
$ ros2 node list
$ ros2 topic list
$ ros2 topic echo /chatter
$ ros2 topic info /chatter

+Key GUI tools:
• rqt_graph: node–topic graph
• rqt: plotting, parameter tuning, more
• rviz2: 3D visualization for robot state, sensors, and TFs

+Here come the Robots!!!
• export TURTLEBOT3_MODEL=burger
• ros2 launch turtlebot3_gazebo turtlebot3_world.launch.py
• ros2 run turtlebot3_teleop teleop_keyboard
